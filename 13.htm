함수의 그래프

1) 이계도함수

함수 y = f(x)에 대해 f′(x)의 도함수를 f(x)의 이계도함수라고 한다.

f(x)의 이계도함수는 등으로 표현한다.


2) 이계도함수와 볼록

함수 f(x)가 어떤 구간에서 이계도함수가 존재할 때
(1) f″(x) > 0 ⇒ f′(x)가 증가 ⇒ 기울기가증가 ⇒ 아래로 볼록
(2) f″(x) < 0 ⇒ f′(x)가 감소 ⇒ 기울기가감소 ⇒ 위로 볼록


3) 이계도함수를 이용하지 않고 위로 볼록과 아래로 볼록을 판단하는 방법

어떤 구간 안에 있는 임의의 a, b에 대해
(1) 이면 y = f(x)는 위로 볼록
(2) 이면 y = f(x)는 아래로 볼록


4) 함수의 극대·극소의 정의

(1) 함수f(x)가 x = a에서 연속이고, x = a의 왼쪽에서는 증가, 오른쪽에서는 감소하면 f(x)는 x = a에서 극대가 되고, f(a)를 극댓값이라고 한다.

(2) 함수 f(x)가 x = a에서 연속이고, x = a의 왼쪽에서는 감소, 오른쪽에서는 증가하면 f(x)는 x = a에서 극소가 되고, f(a)를 극솟값이라고 한다.

(3) 극대가 되는 점(a, f(a))를 극대점, 극소가 되는 점(b, f(b))를 극소점이라 한다.
※ 극댓값, 극솟값을 함께 극값이라 한다.


5) 미분가능한 함수의 극대·극소

y = f(x)가 미분가능한 함수라고 하면
(1) x = a에서 극대 : x = a 왼쪽에서는 f′(x) > 0, x = a 오른쪽에서는 f′ < 0
(2) x = a에서 극소 : x = a 왼쪽에서는 f′(x) < 0, x = a 오른쪽에서는 f′(x) > 0


6) 극값에서의 미분계수

미분가능한 함수f(x)가 x = a에서 극값을 가지면 f″(x) = 0이다. 역은 성립하지 않는다.


7) 이계도함수를 이용한 극값의 판정

이계도함수를 갖는 함수f(x)에 대하여 f″(a) = 0일 때,
(1) f″(a) ＞이면 x = a에서 f(x)는 극소이다.
(2) f″(a) ＜ 0이면 x = a에서 f(x)는 극대이다.
