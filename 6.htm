함수의 극한

함수의 극한 공식
1)함수의 수렴

함수의 수렴 : 함수f(x)에서 x가 a에 한없이 가까워질 때(x ≠ a),f(x)의

값이 일정한 값 α에 한없이 가까워지면 함수f(x)는 α에 수렴한다고 하고 다음과 같이 나타낸다.
또는 x → a일때 f(x) → α  α를 함수 f(x)의 극한값 또는 극한이라고 한다.






2)함수의 발산
함수 f(x)가 수렴하지 않으면 발산이라고 한다.

(1) 양의 무한대로 발산
함수f(x)에서 x의 값이 a에 한없이 가까워질 때, f(x)의 값이 한없이 커지면 f(x)는 양의 무한대로 발산한다고 한다.

 또는 x → a일 때, f(x) → ∞

(2) 음의 무한대로 발산
함수f(x)에서 x의 값이 a에 한없이 가까워질 때, f(x)의 값이 음수이면서 그 절댓값이 한없이 커지면 f(x)는 음의 무한대로 발산한다.

 또는 x → a일 때, f(x) → -∞

3)x → ∞, x → -∞일 때, 함수의 극한
(1) x의 값이 한없이 커질 때 함수f(x)의 값이 일정한 값α에 한없이 가까워지질 때, 함수 f(x)는 α에 수렴한다고 한다.
 또는 x → ∞일때f(x) → α
(2) x의 값이 음수이면서 그 절댓값이 한없이 커질 때 f(x)의 값이 일정한 값β에 한없이 가까워지면, 함수 f(x)는 β에 수렴한다고 한다.
 또는 x → -∞일때f(x) → β
(3) 함수f(x)의 값이 양의 무한대나 음의 무한대로 발산할 때에도 각각 다음과 같은 기호를 써서 나타낸다.
